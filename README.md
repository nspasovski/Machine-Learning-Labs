# Machine-Learning-Lab-Exercises
Machine Learning lab exercises for FINKI

This repository contains a comprehensive collection of machine learning projects and lab exercises completed during my college coursework. It spans foundational statistical models to complex ensemble methods, neural networks, and unsupervised learning pipelines.

---

## 🏆 Featured Project: End-to-End Phishing Websites Classification
**Location:** `/05-classification-final-project`

An end-to-end supervised learning pipeline designed to detect malicious phishing websites using a real-world dataset. 
* **Core Tasks:** Exploratory Data Analysis (EDA), feature selection, data preprocessing, and hyperparameter tuning.
* **Models Evaluated:** Logistic Regression, Decision Trees, Random Forests, Support Vector Classifiers (SVC), and Neural Networks.
* **Metrics Tracked:** Accuracy, Precision, Recall, F1-Score, and ROC-AUC to handle potential data imbalances.
